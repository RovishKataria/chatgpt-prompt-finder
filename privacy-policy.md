# Privacy Policy

**ChatGPT Prompt Finder** does not collect, store, transmit, or share any personal data.

## Data Collection
- No personal data is collected
- No prompts or chat content are stored
- No analytics or tracking is used

## Data Processing
- All processing happens locally in the user's browser
- The extension only reads content already visible on the ChatGPT webpage
- No data leaves the browser

## Network Requests
- The extension makes no external network requests
- No servers, APIs, or third-party services are used

## Permissions
- The extension runs only on chatgpt.com pages
- No additional browser permissions are requested

## Contact
If you have questions about this privacy policy, you can open an issue on the GitHub repository.
